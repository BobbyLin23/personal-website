Building a Modern Personal Website with Nuxt 4

February 1, 2026 8 min read
Nuxt Vue Web Dev

In this article, I'll walk you through building a modern personal website using Nuxt 4, Nuxt UI, and Nuxt Content. We'll cover everything from project setup to deployment.

Why Nuxt 4?

Nuxt 4 brings significant improvements in performance, developer experience, and architecture. The new rendering engine provides faster builds and improved SSR capabilities.

"Nuxt 4 represents a major leap forward in the Vue ecosystem, making full-stack development more accessible and performant than ever before."

Getting Started

First, let's create a new Nuxt project with the latest version:

npx nuxi@latest init my-website
cd my-website
pnpm install

Project Structure

Here's the recommended project structure for a personal website:

my-website/
├── app/
│   ├── components/
│   ├── composables/
│   ├── layouts/
│   ├── pages/
│   └── app.vue
├── content/
│   ├── blog/
│   └── weekly/
├── public/
└── nuxt.config.ts

Setting Up Nuxt Content

Nuxt Content v3 provides a powerful way to manage your markdown content. Install it with:

pnpm add @nuxt/content

Then add it to your nuxt.config.ts:

export default defineNuxtConfig({
  modules: ['@nuxt/content'],
  content: {
    highlight: {
      theme: 'github-dark'
    }
  }
})

Styling with Nuxt UI

Nuxt UI provides a comprehensive set of components built on top of Tailwind CSS. Key benefits include:

  • Pre-built, accessible components
  • Dark mode support out of the box
  • Customizable theme system
  • Responsive design utilities

Deployment

Deploy your website to Vercel, Netlify, or any other platform that supports Node.js. Nuxt 4 supports multiple rendering strategies including SSR, SSG, and ISR.

That's it! You now have a solid foundation for your personal website. In the next article, we'll explore adding internationalization and RSS feed support.
